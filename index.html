Selenium Automation Bootcamp Roadmap

This is the complete roadmap of our selenium automation bootcamp. We will be starting with fundamentals of HTML & CSS which are the building blocks for any websites. Then we gradully move towards Java/Python fundamentals & OOPs concepts

Folks who has enrolled for API automation, we will be starting after UI automation completion and at the end we will be having framework and others tools sessions to cover them in details

1

HTML & CSS Fundamentals

Here, we will be exploring the basic building blocks of any website ie. HTML & CSS which helps to automate any website more efficiently and effectively

2

Java / Python Fundamentals

Here, we will be exploring the details of Python or Java fundamentals. We will be covering everything from scratch with full details that helps to gain confidence in your guys.

3

Java / Python OOPs concepts

Once we complete the fundamentals of Java / Python, we will be starting with OOPs concept of Java / Python

4

UI Automation using Selenium

After gaining the complete knowledge of HTML, CSS, Java/Python we will be starting UI automation using Selenium. Here, we will be exploring everything in and out of Selenium tools in full depth

5

Git & GitHub

Here, we will be covering about one of the most important tools among the programmers community ie. Git & GitHub.

6

Creation of UI automation framework

After completing Selenium part, we will be starting with development of automation framework which is one of the most import part from the point of interview

7

API automation using Java/Python (along with Postman)

Here, we will be starting with API automation for the folks who enrolled for API automation. We will be covering everything about API testing (manual & automation) from scratch.

8

Assignments, Projects & Resume

Finally, we will be having a small mock test and then we start with projects & resume preparation
